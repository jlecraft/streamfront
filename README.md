# streamfront

A Python GUI front-end for [streamlink](https://streamlink.github.io) that lets you launch streams with a single click.

## Features

- One-click stream launching via streamlink and VLC
- Live status indicators and viewer counts for live channels
- Streamlink output log embedded in UI
- Channel list loaded from a plain text file — easy to edit
- Desktop shortcut support via `launch.bat`

## Requirements

- Python 3.10+
- [Streamlink](https://streamlink.github.io/install.html)
- [VLC media player](https://www.videolan.org/vlc/)

## Quick Start

See **[install.md](docs/install.md)** for full step-by-step setup instructions.

```
pip install -r requirements.txt
python init.py
python main.py
```

## Channel List

Edit `channels.txt` to add your channels — one Twitch login per line, with an optional display name separated by a space:

```
# Lines starting with # are ignored
grian
goodtimeswithscar Scar
impulsesv Impulse
```
## Configuration

`config.ini` is generated by `init.py` with all options documented inline. The two optional sections are:

```ini
[player]
path = C:\Path\To\Player.exe

[twitch]
client_id = your_client_id
client_secret = your_client_secret
```

Twitch API credentials are free — see Step 7 of [install.md](docs/install.md) for setup instructions.
